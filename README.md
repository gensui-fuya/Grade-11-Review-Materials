# 📚 Study Repository · 开源学习资料库

> 一个面向高中阶段的系统化学习资料整理仓库  
> 持续更新 · 结构清晰 · 面向考试 · 践行开源精神

---

## 🌱 项目简介

本仓库用于整理和归档高中阶段的学习资料，包括但不限于：

- 📘 化学
- 🧬 生物
- 📐 物理
- 📊 数学
- 🧠 其他学科与专题总结

目标不是简单堆积笔记，而是：

- 建立结构化知识体系  
- 提炼高频考点  
- 结合真题分析命题逻辑  
- 形成可复习、可迭代、可持续优化的学习资料库  

---


> 未来会根据学习进度持续新增内容与专题。

---

## 📘 内容特点

### 🔥 高频考点提炼
基于校内月考 / 期中 / 期末考试趋势整理。

### 📊 表格化总结
核心概念用结构表格呈现，便于快速复习。

### 🧠 逻辑导向
强调“理解原理”而不是死记硬背。

### 📈 可扩展结构
每个学科都可以继续按：

- 章节
- 专题
- 高频题型
- 易错点
- 综合训练

进行模块化扩展。

---

## 🚀 如何使用

1. 按学科进入对应文件夹
2. 按章节查阅知识点
3. 结合习题进行练习
4. 可 Fork 后自行修改完善
